# shoutme
Library to log message to the shell.

1. Install 

```
npm install shoutme
```

2. Usage

```
import shoutme from 'shoutme';


```

3. Development

```
npm run dev
```

This watches files and compiles them

```
npm start
```

To test the library in the terminal.

4. Run tests

```
npm test
```

`npm run dev` should be running to watch and recompile files.
